# RayyanAnsari-DataAnalyst-Internship-Portfolio
# 📊 ApexPlanet Data Analytics Internship Portfolio

## 👨‍💻 Project Overview

Welcome to my **Data Analytics Internship Portfolio**, completed as part of the **60-Day Data Analytics Internship** at **ApexPlanet Software Pvt. Ltd.**

This repository showcases all the projects completed during the internship, covering the complete data analytics workflow—from data cleaning and exploratory data analysis to interactive dashboard development, statistical validation, and business storytelling.

Each task demonstrates practical problem-solving skills using industry-standard tools and technologies.

---

# 🎯 Internship Objectives

The objectives of this internship were to:

- Understand and clean real-world datasets.
- Perform Exploratory Data Analysis (EDA).
- Answer business questions using SQL.
- Create interactive dashboards using Power BI.
- Generate meaningful business insights through data storytelling.
- Validate findings using statistical methods.
- Build a professional GitHub portfolio showcasing all completed work.

---

# 📁 Internship Tasks

| Task | Title | Description |
|------|-------|-------------|
| Task 1 | Data Immersion & Wrangling | Cleaned and transformed raw datasets, handled missing values, duplicates, and prepared analysis-ready data. |
| Task 2 | Exploratory Data Analysis & Business Intelligence | Performed EDA, created visualizations, answered business questions using SQL, and designed dashboard mock-ups. |
| Task 3 | Deep-Dive Analysis & Interactive Dashboarding | Built an interactive Power BI dashboard and analyzed customer, category, and regional performance using KPIs. |
| Task 4 | Data Storytelling & Statistical Validation | Created a business presentation, performed hypothesis testing, and summarized insights into actionable recommendations. |
| Task 5 | Capstone Integration & Portfolio Finalization | Compiled all internship work into a professional GitHub portfolio and documented the overall learning experience. |
